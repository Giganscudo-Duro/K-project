# Vim で vue 拡張子を弄る
以下のコマンドのいずれかを実行。

- Vundle を利用
```sh
Plugin 'posva/vim-vue'
```

- Pathogen を利用
```sh
cd ~/.vim/bundle && \
git clone https://github.com/posva/vim-vue.git
```

- プラグインマネージャを使わない
```sh
$ git clone https://github.com/posva/vim-vue.git ~/.vim/pack/plugins/start/vim-vue
```
