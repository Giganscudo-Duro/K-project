# 必要なパッケージのインストール

1. requests  
PythonでRest API操作を簡単に行うためのライブラリ。

2. requests-oauthlib  
PythonでOAuth認証を簡単に行うためのライブラリ。

以下のコマンドを実行
```sh
# pip install requests requests-oauthlib
```
