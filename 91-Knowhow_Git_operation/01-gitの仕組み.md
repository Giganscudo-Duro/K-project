# イメージ図


git 関係でよく使うコマンド `git add`、`git commit`、`git push` のやってることは、以下の通り。  

| コマンド     | やってること                                                |
|--------------|-------------------------------------------------------------|
| `git add`    | WorkSpace で編集したファイルを、Index に登録する。          |
| `git commit` | Index に登録されたファイルを、Local Repository に登録する。 |
| `git push`   | Local Repository の内容を、Remote Repository に同期する。   |

```txt
    +- Local Machine ------------------------------+              +- Git Hub ------------------------------------+
    |                                              |              |                                              |
    |     +- Local Repository ---------------+     |              |     +- Remote Repository --------------+     |
    |     |                                  |     |              |     |                                  |     |
    |     |   +- Branch-A ---------------+   |     |              |     |   +- Branch-A ---------------+   |     |
    |     |   |                          |   |     |              |     |   |                          |   |     |
    |     |   +--------------------------+   |     |              |     |   +--------------------------+   |     |
    |     |   +- Branch-B ---------------+   |     |  git push    |     |   +- Branch-B ---------------+   |     |
    |     |   |                          |--------------------------------->|                          |   |     |
    |     |   +--------------------------+   |     |              |     |   +--------------------------+   |     |
    |     +----------------------------------+     |              |     +----------------------------------+     |
    |                            ↑                |              +----------------------------------------------+
    |                            ↑ git commit     |
    |                            ↑                |
    |     +- Index(StagingArea) -↑----------+     |
    |     |                      ↑          |     |
    |     |   +- Branch-B ---------------+   |     |
    |     |   |                          |   |     |
    |     |   +--------------------------+   |     |
    |     |                                  |     |
    |     +----------------------------------+     |
    |                            ↑                |
    |                            ↑ git add        |
    |                            ↑                |
    |     +- Work Space ---------↑----------+     |
    |     |                      ↑          |     |
    |     |   +- Branch-B ---------------+   |     |
    |     |   |                          |   |     |
    |     |   +--------------------------+   |     |
    |     |                                  |     |
    |     +----------------------------------+     |
    |                                              |
    +----------------------------------------------+
```
